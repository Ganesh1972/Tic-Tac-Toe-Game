# ❌⭕ Tic Tac Toe Game

A simple and fun web-based **Tic Tac Toe** game built using **HTML**, **CSS**, and **JavaScript**.  
Play against a friend on the same device and test your strategy!

---

## 🎮 Features

- Two-player mode (Player X vs Player O)
- Automatic turn switching
- Win detection and game-over message
- Option to restart the game
- Visually appealing and responsive design

---

## 🛠️ Technologies Used

- **HTML** – For structuring the game layout
- **CSS** – For styling and layout
- **JavaScript** – For game logic and interactions

---

## 📁 Project Structure

| File               | Description                                  |
|--------------------|----------------------------------------------|
| `index.html`       | Contains the game layout and elements        |
| `style.css`        | Handles all the styles and animations        |
| `script.js`        | Implements the core game logic               |
